## 자바스크립트의 데이터 타입은 크게 두 가지로 분류된다.

### 기본형 (primitive type)

- number / 숫자
- string  / 문자열
- boolean / 불리언
- null
- undefined
- ...
- Symbol (ES6)

### 참조형 (reference type)

- object
- Array
- Function
- Date
- RegExp
- (ES6)
  - Map
  - WeakMap
  - Set
  - WeakSet
  - ...
